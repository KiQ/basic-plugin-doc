タイマー機能¶

Timer とは、時間経過を管理する機能です。

例えば、ポイント0～60までをカウント間隔10秒でカウントしていき、10分(600秒)後にカウントがポイントが最大になるという表現ができます。

タイマーの開始時と終了時には Content を付与することもできます。 これを用いて、「キャラクターA」を使用した時点から1時間経過しないと再使用できなくする機能や、5時間経過後に「キャラクターB」を付与するといったことができます。

タイマーの設定例は Timer マスタ を参照してください。

なお、タイマーは以下のライフサイクルをもちます。

  1. timerCreate で作成されたタイマーはカウント状態(RUNNING)となり、カウント用のポイントを持ちます。タイマー開始時に Content を付与する設定がされていた場合、プレイヤーに直接付与を行います、
  2. タイマーは、ポイントが Timer マスタ で定義された最大値に到達するまで、カウントを行います
  3. timerGet (または timerGetList )でポイントが最大値に達したことを確認します
  4. timerClose でタイマーを終了させます(FINISHED)。タイマー終了時に Content を付与する設定がされていた場合、プレイヤーに直接付与を行います。
  5. 不要となったタイマーを timerDelete で削除します

提供するAPI¶

API 名 説明
timerCreate タイマーを新規で作成します
timerClose リクエストしたタイマーを終了させます
timerDelete リクエストしたタイマーを削除します
timerGet リクエストしたタイマーの状態を取得します
timerGetList 現在存在しているタイマー一覧を取得します
timerModifyByContent 特定のコンテンツ効果をタイマーに適応させ状態を変動させます
