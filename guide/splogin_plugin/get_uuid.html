Uuid の取得¶

プレイヤー は、初回接続時に Uuid の取得をおこないます。

このためのAPIが、 sploginGetNonce と sploginGetUuid です。

sploginGetUuid をコールするときに 端末の情報をつけて送信します。

提供するAPI¶

API 名 説明
sploginGetNonce Uuid を取得する際に使用する Nonce を取得します。
sploginGetUuid Nonce を使って Uuid を取得します。
