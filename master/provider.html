Provider マスタ¶

Provider マスタでは、一定時間毎に指定した Content の増加を行うための定義を行います。

以下にマスタの持つ属性値を記載します。

Provider マスタの持つ属性値¶

  • data_version

    定義を行うマスタのバージョン

    型 : 文字列

    必須 : yes

    入力可能文字数 : 1-32

    備考 : DataVersion については DataVersion を参照

  • provider_config_id

    Provider 設定の 識別子

    型 : 整数値

    必須 : yes

    範囲 : 0-9999

  • content_id

    増加を行う Content の 識別子

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

  • add_amount

    対象 Content の増加量

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

  • init_amount

    Provider 開始時のポイント初期値

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

    備考 : max_point を超えない範囲

  • max_amount

    Provider がカウントするポイントの最大値

    型 : 整数値

    必須 : yes

    範囲 : 0-99999( Content の max_amount を超えない範囲)

  • count_interval

    1ポイントカウントするのにかかる秒数

    型 : 整数値

    必須 : yes

    範囲 : 1-99999
