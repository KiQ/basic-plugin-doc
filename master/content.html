Content マスタ¶

Content マスタでは、 BasicPlugin の基礎となるデータを定義します。

プレイヤーに見える「お金」「アイテム」「キャラクター」「武器」「スタミナ」といったデータから、内部データになるような「キャラクターステータス」「フラグ」などを Content マスタとして定義します。

以下にマスタの持つ属性値を記載します。

Content マスタの持つ属性値¶

  • data_version

    定義を行うマスタのバージョン

    型 : 文字列

    必須 : yes

    入力可能文字数 : 1-128

    備考 : DataVersion については DataVersion を参照

  • content_id

    Content の 識別子

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

  • content_type

    Content の種別

    型 : 文字列

    必須 : yes

    使用可能文字： [ 0-9 ] [ a-z ] [ A-Z ] および [ _ ]

    入力可能文字数 : 1-16

  • description

    Content の説明文

    型 : 文字列

    必須 : yes

    入力可能文字数 : 1-2048

  • content_class

    Content の管理ルールの定義

    型 : 文字列

    必須 : yes

    備考 : 定義可能な content_class については ContentClass 一覧 を参照

  • max_amount

    Inventory 内に所持できる最大値

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

    備考 : Inventory に追加、加算された際にこの値を超えると丸め処理が行われ、 max_amount を超えない

  • system_effect_type

    Content を使用することで特殊なシステム効果を適用するための定義

    型 : 文字列

    必須 : no

    備考 : 定義可能な system_effect_type については SystemEffectType 一覧 を参照
