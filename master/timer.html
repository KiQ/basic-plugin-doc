Timer マスタ¶

Timer マスタでは、タイマーの初期値や最大値、ポイントのカウント間隔、タイマーの開始時と終了時に付与する Content 情報を定義します。

Timer を作成する際に、 Timer マスタの timer_config_id を指定することで、定義された属性値が適用されます。

以下にマスタの持つ属性値を記載します。

Timer マスタの持つ属性値¶

  • data_version

    定義を行うマスタのバージョン

    型 : 文字列

    必須 : yes

    入力可能文字数 : 1-32

    備考 : DataVersion については DataVersion を参照

  • timer_config_id

    Timer 設定の 識別子

    型 : 整数値

    必須 : yes

    範囲 : 0-9999

  • init_point

    開始時のポイント初期値

    型 : 整数値

    必須 : yes

    範囲 : 0-99999( max_point を超えない範囲)

  • max_point

    カウントするポイントの最大値

    型 : 整数値

    必須 : yes

    範囲 : 0-99999

  • count_interval

    1ポイントカウントするのにかかる秒数

    型 : 整数値

    必須 : yes

    範囲 : 1-99999

  • begin_add_content_id

    開始時に付与を行う Content の 識別子

    型 : 整数値

    必須 : no

    範囲 : 0-99999

    備考 : 開始時に付与できる Content は 99 個まで

  • begin_add_amount

    開始時に付与を行う|defContent| の個数

    型 : 整数値

    必須 : no

    範囲 : 0-99999

    備考 : Content の max_amount を超えない範囲

  • end_add_content_id

    終了時に付与を行う Content の 識別子

    型 : 整数値

    必須 : no

    範囲 : 0-99999

    備考 : 終了時に付与できる Content は 99 個まで

  • end_add_amount

    終了時に付与を行う Content の個数

    型 : 整数値

    必須 : no

    備考 : Content の max_amount を超えない範囲

一定時間後にプレイヤーに 「武器」 を付与する Timer の設定例¶

「素材A」と引き換えに、1時間後「武器A」をプレイヤーに付与する Timer の設定例です。

1． 「素材A」を消費して「引換券1」を付与する Exchange を実行

2． タイマーを生成し、終了時に「引換券2」を付与

3． 「引換券1」,「引換券2」を消費して「武器A」を付与する Exchange を実行
