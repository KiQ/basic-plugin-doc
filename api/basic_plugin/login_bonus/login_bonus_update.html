loginBonusUpdate¶

LoginBonus マスタに応じてログインカウント日数を更新するAPIです。

日数に対応する LoginBonus 付与 Content が定義されていた場合、 Present 状態の Content を Inventory に付与します。

loginBonusUpdate(LoginBonusUpdateRequest) => LoginBonusUpdateResult

Requests¶

  • None

Returns¶

  • success (bool)

    APIの成否

  • continueCount (int)

    更新後のログインカウント数

  • totalContinueCount (int)

    累計ログインカウント数

  • addPresentList (list<Present>)

    付与した Present のリスト

    備考 : 付与された Present がない場合、空のリストを返す

Examples¶

リクエストの例¶

{
}

レスポンスの例¶

{
    success: true,
    continueCount: 5,
    addPresentList: [
        {
            inventoryId: 1000,
            contentId: 10,
            amount: 10,
            description: ''
        },
        {
            inventoryId: 2000,
            contentId: 20,
            amount: 10,
            description: '',
            expireDate: 1460041200000  /* Epoch milliseconds : 2016/4/8 0:0:0 */
        }
    ]
}

Errors¶

API固有のエラーは定義されていません。
