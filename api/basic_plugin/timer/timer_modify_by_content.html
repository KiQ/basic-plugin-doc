timerModifyByContent¶

プレイヤー Inventory にある SYSTEM_EFFECT_TYPE を持った Content を消費して、 Timer のポイントを変更するAPIです。

timerModifyByContentRequest(TimerModifyByContentRequest) => TimerModifyByContentResult

Requests¶

  • statusId (int)

    ポイントを変更した Timer の 識別子

  • inventoryId (int)

    使用する Content の Inventory の 識別子

Returns¶

  • success (bool)

    APIの成否

  • updateTimer (Timer)

    ポイントを変更した後の Timer の状態

  • updateInventoryElement (list<InventoryElement>)

    更新された Inventory のリスト

    備考 : 更新が行われなかった場合、空のリストを返す

  • deleteInventoryElement (list<InventoryElement>)

    削除された Inventory のリスト

    備考 : 削除が行われなかった場合、空のリストを返す

Examples¶

リクエストの例¶

{
    statusId: 1,
    inventoryId: 1000
}

レスポンスの例¶

{
    success: true,
    updateTimer: {
        statusId: 1,
        configId: 100,
        state: 'FINISHED'
        point: 500,
        lastUpdate: 1459436400000,  /* Epoch milliseconds : 2016/4/1 0:0:0 */
        countInterval: 100,
        maxPoint: 500
    },
    updateInventoryElement: {
        inventoryId: 1000,
        contentId: 10,
        contentType: 'type-01',
        amount: 9,
        description: 'Effects timer',
        isPayment: false
    }
}

Errors¶

エラーコード 説明
20700 不正な statusId がリクエストされました
20200 不正な inventoryId がリクエストされました
20705 効果を適用できませんでした. エラーメッセージを参照してください
