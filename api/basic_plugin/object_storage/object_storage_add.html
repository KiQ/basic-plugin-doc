objectStorageAdd¶

ObjectStorage にキーとバリューのセットを保存します。

objectStorageAdd(ObjectStorageAddRequest) => ObjectStorageAddResult

Requests¶

  • key (string)

    プレイヤー毎に一意な、データを示すキー

  • value (binary)

    ObjectStorage に保存するデータ

  • hash (binary)

    value のハッシュ値

Returns¶

  • success (bool)

    APIの成否

Examples¶

リクエストの例¶

{
    key: 'key-A',
    value: /* binary */
    hash: /* binary */
}

レスポンスの例¶

{
    success: true
}

Errors¶

エラーコード 説明
20801 ObjectStorage 内に同一の key が存在します
20802 hash の確認に失敗しました
