Inventory の加減算方法¶

Inventory 内の Content に対する加算、減算の方法は Content マスタの content_class 属性の設定によって異なります。

以下に、各 Content 毎の加算、減算の方法について示します。

Countable¶

  • Inventory 内での加算方法

    加算が行われる際は対象の Content の 識別子 を持つレコードに対して加算を行います。

    対象の Content の 識別子 を持つレコードがない場合は追加されます。

  • Inventory 内での減算方法

    減算が行われる際は対象の Content の 識別子 を持つレコードに対して減算を行います。

Uncountable¶

  • Inventory 内での加算方法

    加算が行われる際は Inventory の末尾にレコードを追加します。

  • Inventory 内での減算方法

    減算が行われる際は追加されたレコード順に減算します。

    減算対象となる Inventory の 識別子 を直接指定することもできます。

Payment¶

  • Inventory 内での加算方法

    加算が行われる際は Inventory の末尾にレコードを追加します。

  • Inventory 内での減算方法

    減算が行われる際は古いレコード順に減算します。

    減算対象となる Inventory の 識別子 を直接指定することもできます。
