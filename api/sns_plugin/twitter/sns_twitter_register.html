accountRegisterTwitter¶

Twitterアカウントとの連携を登録するAPIです。

利用時は、端末上でTwitter認証を行い、アクセストークンとトークンシークレットを取得してください。

accountRegisterTwitter(AccountRegisterTwitterRequest) => AccountRegisterTwitterResult

Requests¶

  • account (String)

    SNS でのアカウント 識別子

  • accessToken (String)

    Twitter API利用のためのアクセストークン

  • accessTokenSecret (String)

    Twitter API利用のためのトークンシークレット

  • snsProfile (optional, String)

    SNSでの表示名など。記録内容、書式についてはAPIでは規定しない。

Returns¶

  • success (bool)

    APIの成否

  • errorCode (optional, int)

    API失敗時のエラーコード

  • errorMessage (optional, String)

    API失敗時のエラーメッセージ

Examples¶

リクエストの例¶

{
    account: '12345678',
    accessToken: 'token12345678',
    accessTokenSecret: 'tokenSecret12345678',
    snsProfile: '{...}'
}

レスポンスの例¶

{
    success: true
}
