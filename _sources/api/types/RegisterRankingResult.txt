RegisterRankingResult
====================

====================

* success (bool)
    APIの成否
* rankingId (optional, int)
    |defRanking| の |defId|
* registerRankingResultState (optional, <:doc:`RegisterRankingResultState`>)
    登録結果
* errorCode (optional, int)
    API失敗時のエラーコード
* errorMessage (optional, string)
    API失敗時のエラーメッセージ
