LoginBonus¶

LoginBonus とは、設定された切り替え時刻を基準にログインカウントを行っていき、カウント数に見合った Content をプレイヤーにプレゼントとして付与する機能です。

例えば、「05:00」以降にログインした場合カウント数を 1加算し、カウント数が10だった場合は「武器A」をプレゼントするといった表現ができます。

ログインカウント数は設定された期間でリセットが行われ、設定の仕方で週間、月間といった表現もできます。

また、プレゼントする Content は複数種類にすることができます。

カウントの切り替え時刻、ログインカウントの期間、プレゼントする Content は LoginBonus マスタで設定します。

LoginBonus マスタの設定例は LoginBonus マスタ を参照してください。

提供するAPI¶

API 名 説明
loginBonusUpdate ログインカウントを行い、カウント数に見合った報酬を付与します
loginBonusGetList 現在期間中の LoginBonus の内容を取得します
