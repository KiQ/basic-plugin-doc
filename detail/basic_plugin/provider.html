Provider¶

Provider とは、一定時間毎に Content を増加することができる機能です。

例えば、対象とする Content を「スタミナ」とした場合 最大スタミナ24のプレイヤーが12までスタミナを消費していた時、スタミナが5分ごとに1ずつ増加していき1時間後にはスタミナが24になる、といった表現できます。

加算対象の Content や初期ポイント、最大ポイント、カウント間隔などは Provider マスタで設定します。

Provider マスタの設定については、Provider マスタ を参照してください。

提供するAPI¶

API 名 説明
providerCreate Provider を新規で作成します
providerUpdate リクエストした Provider を更新します
providerDelete リクエストした Provider を削除します
