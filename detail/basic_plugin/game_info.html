GameInfo¶

GameInfo とは、予め設定されているゲームに関する情報を、キーをリクエストすることで取得できる機能です。

例えば、以下の様なゲーム情報が格納されているとします。

{
    key: 'data_version',
    value: 'versionA'
},
{
    key: 'maintenance',
    value: '2016/09/01 12:34:56'
}

key をリクエストしこれらを取得することで、メンテ時間中のゲームの動作を変える手掛かりにするといった使い方ができます。

GameInfo マスタについては GameInfo マスタ を参照してください。

提供するAPI¶

API 名 説明
gameInfoGet リクエストしたキーとペアの GameInfo を取得します
gameInfoGetList GameInfo を一覧で取得します
